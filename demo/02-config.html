  • Slide 1

    This is a deckard slide, inside a "set" of deckard slides.

    This page contains some Deckard config overrides, which are specified by defining Deckard before including the script, like so:

    <script>
      Deckard = { transition:'fade' };
    </script>
    <script src="deckard.js"></script>

  • Slide 3

    This is another deckard slide.

    I have a different animation style to the default. The change in animation was specified declaritively in my HTML by adding this attribute:
    data-deckard-transition="fall"

    Note the change in direction as you navigate into and out of this slide forwards and backwards.
